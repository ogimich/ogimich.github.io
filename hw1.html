Homework 1 - Specifications

1. Create a web page, using HTML5, with the heading CNIT 133 Homework. The contents of this page can be a list containing the homework assignment numbers which will become a link when you complete and upload the corresponding assignments. You should list all homework assignments (Homework 2 through Homework 8). Homework 1 will not be listed as this web page represents Homework 1. Embellish this page with some CSS(backgroundcolor,and/or background image, and/or different font type/size/color, you can think about making each link look like a button, etc.)

2. Create the files hw2.html through hw8.html. Make sure that you modify the title, meta description for each of those pages accordingly and, for now, just include a heading saying something as "Coming Soon!" Save each file to your computer and then upload all the files (hw1.html / or index.html and also hw2.html, hw3.html....hw8.html) to your public_html directory in Hills or to the web server you want to use. You will later create the content of these pages throughout the semester

3. I suggest that you validate your Homework 1 page for HTML5 and CSS.

4. Remember to use HTML5 and also the two important meta tags (charset and description) If you need help in fixing very difficult to find validation errors, as noted from the validator, then you should post your doubt in the forum.

To submit your homework:

1. Once you upload your Homework 1 page and all the other pages (hw2.html up to hw8.html) to the Hills server, make sure you can view the pages in a web browser.

 2. Highlight the URL of your base Homework 1 page in the browser address bar and copy it. 

3. Paste the URL of your Homework 1 page in the box provided to submit your assignment 

4. Click the "Submit assignment" button to submit your assignment.

Checklist

- your page opens in the browser with no errors (NOT FOUND or FORBIDDEN) - 2 points

 - all your links are working (Homework 2 link is opening hw2.html, Homework 3 link is opening hw3.html, etc.) - 3 points

 - your HTML5 and CSS code are valid - 5 points
